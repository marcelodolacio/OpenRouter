# OpenRouter

Projeto desenvolvido pra ir sempre atualizando a llm que estamos usando no Openrouter, podendo ter vários tipos de filtros, como preço, throughput ou TPS.

Tecnologias utilizadas:


- OpenAI
- Node.js
- Fastify
- OpenRouter

Para rodar o projeto:


- Comando npm ci pra restabelecer as dependencias.
- Comando npm start pra rodar o projeto.
- Criar um arquivo .env com suas secrets de acordo com o exemplo.
